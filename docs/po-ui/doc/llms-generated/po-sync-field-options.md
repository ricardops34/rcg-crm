# PoSyncFieldOptions

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-sync-field-options

Interface que irá mapear as configurações dos campos do `PoSyncSchema`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `local` | `boolean` | sim | Propriedade que informa se o campo será apenas para armazenamento local |
| `name` | `string` | não | Propriedade que informa o nome do campo. |
