# PoPageDynamicDetailBeforeRemove

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-page-dynamic-detail-before-remove

Definição da estrutura de retorno da url ou método executado através da
propriedade `beforeRemove`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `allowAction` | `boolean` | sim | Define se deve ou não executar a ação de exclusão (*remove*) |
| `newUrl` | `string` | sim | Nova rota para navegação que substituirá a definida anteriormente em `remove`. |
