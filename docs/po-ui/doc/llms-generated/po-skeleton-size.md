# PoSkeletonSize

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-skeleton-size

Define os tamanhos disponíveis para o componente `po-skeleton`.

## Valores

| Valor | Descrição |
|---|---|
| `xs` | Tamanho extra pequeno |
| `sm` | Tamanho pequeno |
| `md` | Tamanho médio |
| `lg` | Tamanho grande |
| `xl` | Tamanho extra grande |
| `xxl` | Tamanho extra extra grande |
