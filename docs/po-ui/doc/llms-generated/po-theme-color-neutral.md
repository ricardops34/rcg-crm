# PoThemeColorNeutral

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-theme-color-neutral

Interface para as cores neutras do tema.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `dark` | `{
    '70'?: string;
    '80'?: string;
    '90'?: string;
    '95'?: string;
}` | sim | Cores Neutrals do tipo 'dark'. |
| `light` | `{
    '00'?: string;
    '05'?: string;
    '10'?: string;
    '20'?: string;
    '30'?: string;
}` | sim | Cores Neutrals do tipo 'light'. |
| `mid` | `{
    '40'?: string;
    '60'?: string;
}` | sim | Cores Neutrals do tipo 'mid'. |
