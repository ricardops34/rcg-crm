# PoLookupResponseApi

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-lookup-response-api

Interface que representa a estrutura de resposta de uma coleção de itens.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `hasNext` | `boolean` | não | Indica se existe uma próxima página com mais registros para aquela coleção de itens. |
| `items` | `Array<object>` | não | Lista de itens retornados. |
