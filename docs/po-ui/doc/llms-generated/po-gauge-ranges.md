# PoGaugeRanges

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-gauge-ranges

Interface que define os alcances das cores do gauge.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `color` | `string` | sim | Determina a cor do alcance. As maneiras de customizar o *preset* padrão de cores são: |
| `from` | `number` | sim | Alcance inicial da cor. O valor padrão é 0. |
| `label` | `string` | sim | O texto para a legenda do alcance. |
| `to` | `number` | sim | Alcance final da cor. O valor padrão é 100. |
