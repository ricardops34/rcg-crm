# PoTableRowTemplateArrowDirection

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-table-row-template-arrow-direction

Define a posição da arrow que expande o * row template* na tabela, será exibida na esquerda ou direita.

## Valores

| Valor | Descrição |
|---|---|
| `Left` | Posiciona a *arrow* na esquerda. (Padrão) |
| `Right` | Posiciona a *arrow* na direita |
