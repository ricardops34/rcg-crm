# PoTimepickerModelFormat

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-timepicker-model-format

Enum* que define o padrão de formatação do model de saída do timepicker.

## Valores

| Valor | Descrição |
|---|---|
| `HourMinute` | Formato básico `HH:mm` (ex: `14:30`). |
| `HourMinuteSecond` | Formato com segundos `HH:mm:ss` (ex: `14:30:00`). |
