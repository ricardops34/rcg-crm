# PoSwitchLabelPosition

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-switch-label-position

Enum para posicionar o label do valor do po-switch.

## Valores

| Valor | Descrição |
|---|---|
| `Right` | Posiciona o label do lado esquerdo do switch. |
| `Left` | Posiciona o label do lado direito do switch. |
