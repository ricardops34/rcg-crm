# PoNetworkStatus

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-network-status

Classe responsável por identificar a conexão de rede disponível no dispositivo.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `status` | `boolean` | não | Retorna se o dispositivo está conectado na rede. |
| `type` | `PoNetworkType` | não | Retorna o tipo de conexão do dispositivo. |
