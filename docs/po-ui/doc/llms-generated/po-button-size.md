# PoButtonSize

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-button-size

Tamanhos da propriedade `p-size`. A medida `small` está disponível apenas para acessibilidade AA.

## Valores

| Valor | Descrição |
|---|---|
| `Small` | Define a altura do button como 32px. |
| `Medium` | Define a altura do button como 44px. |
| `Large` | Define a altura do button como 56px. |
