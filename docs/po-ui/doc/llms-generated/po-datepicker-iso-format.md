# PoDatepickerIsoFormat

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-datepicker-iso-format

Enum* que define o padrão de formatação das datas.

> Caso um formato padrão seja definido, o mesmo não será mais alterado de acordo com o formato de entrada.

## Valores

| Valor | Descrição |
|---|---|
| `Basic` | Padrão **E8601DAw** (*yyyy-mm-dd*). |
| `Extended` | Padrão **E8601DZw** (*yyyy-mm-ddThh:mm:ss+\|-hh:mm*). |
