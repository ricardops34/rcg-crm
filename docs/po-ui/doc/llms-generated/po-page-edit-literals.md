# PoPageEditLiterals

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-page-edit-literals

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `cancel` | `string` | sim | Label da ação `cancel`. |
| `save` | `string` | sim | Label da ação `save`. |
| `saveNew` | `string` | sim | Label da ação `saveNew`. |
