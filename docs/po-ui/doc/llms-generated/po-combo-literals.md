# PoComboLiterals

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-combo-literals

Interface para definição das literais usadas no `po-combo`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `chooseOption` | `string` | sim | Texto exibido quando o combo estiver vazio. |
| `clean` | `string` | sim | Texto do aria-label do botão de limpar |
| `noData` | `string` | sim | Texto exibido quando não houver itens na lista ou se, a pesquisa do filtro não retornar nenhum item. |
