# PoDividerSize

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-divider-size

Enum para definição da espessura da linha.

## Valores

| Valor | Descrição |
|---|---|
| `small` | A espessura da linha fica com 1px. |
| `medium` | A espessura da linha fica com 2px. |
| `large` | A espessura da linha fica com 4px. |
