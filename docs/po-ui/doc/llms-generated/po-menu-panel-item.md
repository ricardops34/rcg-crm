# PoMenuPanelItem

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-menu-panel-item

Interface para os itens de menu do componente `po-menu-panel`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `action` | `Function` | sim | Ação personalizada para clique do item de menu. |
| `icon` | `string` | não | Ícone para o item de menu, os [ícones aceitos](https://po-ui.io/icons) são os definidos no guia de estilo da PO. |
| `label` | `string` | não | Texto do item de menu. |
| `link` | `string` | sim | Link para redirecionamento no click do item do menu, podendo ser um link interno ou externo. |
