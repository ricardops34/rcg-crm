# PoSkeletonType

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-skeleton-type

Define os tipos visuais disponíveis para o componente `po-skeleton`.

## Valores

| Valor | Descrição |
|---|---|
| `normal` | Tipo padrão com cor neutra clara |
| `primary` | Tipo primário com cor neutra média |
| `content` | Tipo de conteúdo com fundo branco |
