# PoNavbarLiterals

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-navbar-literals

Interface para definição das literais usadas no `po-navbar`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `navbarLinks` | `string` | sim | Texto exibido no item de menu no qual os itens do navbar são agrupados quando está no modo responsivo. |
