# PoProgressSize

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-progress-size

Enum para configurar a expessura (`p-size`) da barra de progresso do componente.

## Valores

| Valor | Descrição |
|---|---|
| `medium` | Tamanho médio com 4px. |
| `large` | Tamanho grande com 8px. |
