# PoTagType

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-tag-type

Define os tipos disponíveis para o `po-tag`.

## Valores

| Valor | Descrição |
|---|---|
| `Danger` | Erro, perigo, problema ou aviso crítico. |
| `Info` | Informativo ou explicativo. |
| `Success` | Confirmação, resultados positivos ou êxito. |
| `Warning` | Aviso ou advertência. |
| `Neutral` | De uso geral, quando os tipos Info, Warning, Success e Danger não atendem a necessidade. |
