# PoPageDynamicEditField

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-page-dynamic-edit-field

Interface dos fields usados para compor o template `po-page-dynamic-edit`.
Herda as definições da interface
[PoDynamicFormField](https://po-ui.io/documentation/po-dynamic-form).

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `duplicate` | `boolean` | sim | Indica se o campo será duplicado caso seja executada a ação de duplicação. |
