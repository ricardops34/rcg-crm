# PoRadioGroupOption

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-radio-group-option

Interface para as ações do componente po-radio-group.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `disabled` | `boolean` | sim | Desabilita o radio. |
| `label` | `string` | não | Texto do radio. |
| `value` | `string | number` | não | Valor do radio. |
