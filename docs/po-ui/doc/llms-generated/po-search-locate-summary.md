# PoSearchLocateSummary

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-search-locate-summary

Interface que define o resumo de localização do filtro `p-filter-locate`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `currentIndex` | `number` | não | Índice atual da ocorrência localizada. |
| `total` | `number` | não | Total de ocorrências encontradas. |
