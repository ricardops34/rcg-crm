# PoProgressStatus

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-progress-status

Enum `PoProgressStatus` para os status de barra de progresso.

## Valores

| Valor | Descrição |
|---|---|
| `Default` | Define o status `default` para a barra de progresso. |
| `Error` | Define o status de `error` para a barra de progresso. |
| `Success` | Define o status de `success` para a barra de progresso. |
