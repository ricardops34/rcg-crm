# PoDialogAlertLiterals

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-dialog-alert-literals

Interface para definição das literais usadas no serviço `po-dialog` para o tipo alerta.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `ok` | `string` | sim | Rótulo do botão de "Ok". |
