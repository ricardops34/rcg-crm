# PoSelectOption

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-select-option

Interface da coleções de itens que deve ser informado no componente po-select

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `label` | `string` | não | Label a ser utilizada nos itens da lista. |
| `value` | `string | number` | não | Valor do objeto que será atribuído ao model. |
