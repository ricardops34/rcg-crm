# PoDynamicFieldType

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-dynamic-field-type

Enum para definição do tipo de campo que será criado dinamicamente.

## Valores

| Valor | Descrição |
|---|---|
| `Boolean` | Valor booleano. |
| `Currency` | Valor numérico que contém casas decimais e milhar. |
| `Decimal` | Valor numérico que contém casas decimais e milhar. |
| `Date` | Valor para data. |
| `DateTime` | Valor para data e hora. |
| `Time` | Utilizado para informar/exibir hora. |
| `Number` | Valor numérico. |
| `String` | Texto. |
| `Upload` | Utilizado para fazer uploads de arquivos. |
