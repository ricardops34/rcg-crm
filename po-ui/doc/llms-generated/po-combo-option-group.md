# PoComboOptionGroup

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-combo-option-group

Interface dos agrupamentos da coleção que será exibida no dropdown do `po-combo`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `label` | `string` | não | Título para cada grupo de opções. |
| `options` | `Array<PoComboOption>` | não | Lista de itens a serem exibidos. |
