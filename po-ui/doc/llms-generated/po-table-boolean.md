# PoTableBoolean

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-table-boolean

Interface que define as colunas booleanas do `po-table`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `falseLabel` | `string` | sim | Define o rótulo para valores `false`. |
| `trueLabel` | `string` | sim | Define o rótulo para valores `true`. |
