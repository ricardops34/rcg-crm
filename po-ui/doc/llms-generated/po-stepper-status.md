# PoStepperStatus

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-stepper-status

Enums* para os status do `po-stepper` quando utilizada a propriedade `p-steps`.

## Valores

| Valor | Descrição |
|---|---|
| `Active` | Define o estado do *step* como ativo. |
| `Default` | Define o estado do *step* como padrão. |
| `Disabled` | Define o estado do *step* como desabilitado. |
| `Done` | Define o estado do *step* como concluído. |
| `Error` | Define o estado do *step* com erro. |
