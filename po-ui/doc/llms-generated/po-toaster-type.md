# PoToasterType

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-toaster-type

Define os tipos possíveis para o `PoToasterComponent`.

## Valores

| Valor | Descrição |
|---|---|
| `Error` | Tipo de toaster para mensagens de erro. |
| `Information` | Tipo de toaster para mensagens informativas. |
| `Success` | Tipo de toaster para mensagens de sucesso. |
| `Warning` | Tipo de toaster para mensagens de atenção. |
