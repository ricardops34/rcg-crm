# PoI18nConfig

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-i18n-config

Interface para a configuração do módulo `PoI18nModule`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `contexts` | `PoI18nConfigContext` | não | Deve ser atribuído a esta propriedade um objeto que contenha os contextos com os |
| `default` | `PoI18nConfigDefault` | sim | Configurações padrões. |
