# PoTableSubtitleColumn

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-table-subtitle-column

Interface para configuração das colunas de legenda do Po-Table.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `color` | `string` | sim | Define a cor do *status*. |
| `content` | `string` | não | Conteúdo que será exibido na coluna da tabela. |
| `label` | `string` | não | Texto que será exibido no rodapé da tabela como legenda. |
| `value` | `string | number` | não | Valor que será usado como referência para exibição do conteúdo na coluna. |
