# PoSkeletonVariant

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-skeleton-variant

Define as variantes visuais do componente `po-skeleton`.

## Valores

| Valor | Descrição |
|---|---|
| `text` | Variante para simular texto |
| `rectangle` | Variante retangular (largura maior que altura) |
| `square` | Variante quadrada (largura igual à altura) |
| `circle` | Variante circular |
