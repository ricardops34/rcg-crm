# PoSelectOptionGroup

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-select-option-group

Interface da coleções de itens em grupo, utilizando uma label para o grupo e as opções do tipo `PoSelectOption`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `label` | `string` | não | Label para denominar o nome do grupo. |
| `options` | `Array<PoSelectOption>` | não | Lista com as opções disponíveis em cada grupo. |
