# PoTableColumnSpacing

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-table-column-spacing

Tipos de espaçamento interno (padding) das células (**p-spacing**) do po-table.

## Valores

| Valor | Descrição |
|---|---|
| `ExtraSmall` | Espaçamento extra pequeno: 0.25rem (vertical) x 0.5rem (horizontal). |
| `Small` | Espaçamento pequeno: 0.5rem (vertical) x 1rem (horizontal). |
| `Medium` | Espaçamento médio: 0.75rem (vertical) x 1rem (horizontal). |
| `Large` | Espaçamento grande: 1rem (vertical) x 1rem (horizontal). |
