# PoChartDataLabel

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-chart-data-label

Interface que define as propriedades de exibição dos rótulos das séries no `po-chart`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `fixed` | `boolean` | sim | Indica se o texto associado aos pontos da série deve permanecer fixo na exibição do gráfico. |
