# PoTimerFormat

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-timer-format

Enum para definição do formato de exibição do timer.

## Valores

| Valor | Descrição |
|---|---|
| `Format24` | Formato de 24 horas (0-23). |
| `Format12` | Formato de 12 horas (1-12) com indicador AM/PM. |
