# PoBreadcrumb

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-breadcrumb

Interface que define o `po-breadcrumb`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `favorite` | `string` | sim | Permite definir uma URL para favoritar ou desfavoritar. |
| `items` | `Array<PoBreadcrumbItem>` | não | Lista de itens do _breadcrumb_. |
| `params` | `object` | sim | Objeto que possibilita o envio de parâmetros adicionais à requisição. |
