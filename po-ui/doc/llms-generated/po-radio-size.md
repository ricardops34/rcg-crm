# PoRadioSize

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-radio-size

Tamanhos da propriedade `p-size`. A medida `small` está disponível apenas para acessibilidade AA.

## Valores

| Valor | Descrição |
|---|---|
| `Small` | Define o radio como 16px. |
| `Medium` | Define o radio como 24px. |
| `Large` | Define o radio como 32px. |
