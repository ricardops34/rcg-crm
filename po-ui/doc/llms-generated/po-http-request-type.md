# PoHttpRequestType

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-http-request-type

Define o método de requisição HTTP.

## Valores

| Valor | Descrição |
|---|---|
| `DELETE` | Método `delete` do protocolo HTTP. |
| `GET` | Método `get` do protocolo HTTP. |
| `HEAD` | Método `head` do protocolo HTTP. |
| `OPTIONS` | Método `options` do protocolo HTTP. |
| `PATCH` | Método `patch` do protocolo HTTP. |
| `POST` | Método `post` do protocolo HTTP. |
| `PUT` | Método `put` do protocolo HTTP. |
