# PoPageComponentsSize

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-page-components-size

Tamanhos da propriedade `p-components-size` para componentes de formulário no template. A medida `small` está
disponível apenas para acessibilidade AA.

## Valores

| Valor | Descrição |
|---|---|
| `Small` | Aplica a medida `small` do componente. |
| `Medium` | Aplica a medida `medium` do componente. |
