# ForceBooleanComponentEnum

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/force-boolean-component-enum

Enum para definição do tipo de componente a ser renderizado.

## Valores

| Valor | Descrição |
|---|---|
| `switch` | Força a renderização de um po-switch |
| `checkbox` | Força a renderização de um po-checkbox |
