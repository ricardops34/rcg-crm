# PoComboFilterMode

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-combo-filter-mode

Define o tipo de busca usado no po-combo.

## Valores

| Valor | Descrição |
|---|---|
| `startsWith` | Verifica se o texto *inicia* com o valor pesquisado. Caso não seja especificado um tipo, será esse o utilizado. |
| `contains` | Verifica se o texto *contém* o valor pesquisado. |
| `endsWith` | Verifica se o texto *finaliza* com o valor pesquisado. |
