# PoListViewLiterals

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-list-view-literals

Interface para definição das literais usadas no `po-list-view`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `hideDetails` | `string` | sim | Rótulo do botão que oculta os detalhes do item. |
| `loadMoreData` | `string` | sim | Rótulo do botão que deve carregar mais resultados. |
| `noData` | `string` | sim | Rótulo exibido quando não existem itens para serem exibidos na lista. |
| `selectAll` | `string` | sim | Rótulo do `checkbox` da opção de selecionar todos. |
| `showDetails` | `string` | sim | Rótulo do botão que exibe os detalhes do item. |
