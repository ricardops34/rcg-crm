# PoDisclaimer

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-disclaimer

Interface que representa o objeto `po-disclaimer`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `hideClose` | `boolean` | sim | Se verdadeiro, oculta o botão para fechar o *disclaimer*. |
| `label` | `string` | sim | Texto de exibição do objeto. |
| `property` | `string` | sim | Nome da propriedade vinculada ao objeto *disclaimer*. |
| `value` | `any` | não | Valor do objeto. |
