# PoLanguage

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-language

Interface para descrição das linguagens disponíveis no sistema.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `description` | `string` | sim | Descrição do idioma |
| `language` | `string` | sim | Código do idioma [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes) |
