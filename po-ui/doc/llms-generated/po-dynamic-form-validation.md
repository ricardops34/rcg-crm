# PoDynamicFormValidation

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-dynamic-form-validation

Estrutura de retorno da validação do formulário.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `fields` | `Array<PoDynamicFormField>` | sim | Lista com as novas definições dos campos. |
| `focus` | `string` | sim | Nome do campo que receberá o foco. |
| `value` | `any` | sim | Objeto contendo os novos valores. |
