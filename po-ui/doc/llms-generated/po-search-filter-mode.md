# PoSearchFilterMode

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-search-filter-mode

Define o tipo de busca usado no `po-search`.

## Valores

| Valor | Descrição |
|---|---|
| `startsWith` | Verifica se o texto *inicia* com o valor pesquisado. |
| `contains` | Verifica se o texto *contém* o valor pesquisado. |
| `endsWith` | Verifica se o texto *finaliza* com o valor pesquisado. |
