# PoCheckboxSize

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-checkbox-size

Tamanhos da propriedade `p-size`. A medida `small` está disponível apenas para acessibilidade AA.

## Valores

| Valor | Descrição |
|---|---|
| `Small` | Define a caixa de seleção como 16px. |
| `Medium` | Define a caixa de seleção comoc24px. |
| `Large` | Define a caixa de seleção comoc32px. |
