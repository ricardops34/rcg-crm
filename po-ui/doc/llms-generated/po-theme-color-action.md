# PoThemeColorAction

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-theme-color-action

Interface para as cores de ação do tema.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `default` | `string` | sim | Cores da Action 'Default'. |
| `disabled` | `string` | sim | Cores da Action de 'disabled'. |
| `focus` | `string` | sim | Cores da Action para 'focus'. |
| `hover` | `string` | sim | Cores da Action para 'hover'. |
| `pressed` | `string` | sim | Cores da Action para 'pressed'. |
