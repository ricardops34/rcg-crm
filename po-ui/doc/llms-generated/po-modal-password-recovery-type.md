# PoModalPasswordRecoveryType

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-modal-password-recovery-type

Enum* para especificação do tipo de recuperação de senha.

## Valores

| Valor | Descrição |
|---|---|
| `All` | Possibilita ao usuário optar por envio via email ou SMS |
| `Email` | Definição para recuperação apenas por email |
| `SMS` | Definição para recuperação apenas por SMS |
