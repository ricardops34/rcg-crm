# PoSearchFilterSelect

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-search-filter-select

Interface que define as opções que serão exibidas no dropdown do `po-search`, ao usar a propriedade `p-filter-select`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `label` | `string` | não | Descrição exibida nas opções da lista. |
| `value` | `Array<string> | string` | não | Valores que serão atribuídos ao `p-filter-keys` |
