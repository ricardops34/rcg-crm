# PoNetworkType

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-network-type

Tipos de rede existentes no dispositivo.

## Valores

| Valor | Descrição |
|---|---|
| `unknown` | Define o tipo de rede como desconhecido (`unknown`). |
| `ethernet` | Define o tipo de rede como `Ethernet`. |
| `wifi` | Define o tipo de rede como `WiFi`. |
| `_2g` | Define o tipo de rede como `2G`. |
| `_3g` | Define o tipo de rede como `3G`. |
| `_4g` | Define o tipo de rede como `4G`. |
| `cellular` | Define o tipo de rede como `cellular`. Isso acontece na utilização dos navegadores |
| `none` | Define o tipo de rede como `none`. |
