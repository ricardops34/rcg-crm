# PoWidgetAvatar

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-widget-avatar

Interface para definição do avatar no `po-widget`.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `customTemplate` | `TemplateRef<any>` | sim | Permite a criação de template customizado para o avatar |
| `size` | `string` | sim | Tamanho de exibição do componente `po-avatar`. |
| `src` | `string` | sim | Fonte da imagem que pode ser um caminho local (`./assets/images/logo-black-small.png`) |
| `widthCustomTemplate` | `string` | sim | Define a largura em porcentagem do `customTemplate`. |
