# PoPageActionsLayout

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-page-actions-layout

Define os layouts de exibição das ações no cabeçalho do `po-page-default`.

> Compatível com todos os valores de `PoPageHeaderType`.

## Valores

| Valor | Descrição |
|---|---|
| `default` | Exibe as ações como botões (até 3 em desktop e 2 em mobile), agrupando as demais no *dropdown*. |
| `dropdown` | Agrupa todas as ações exclusivamente dentro do menu *dropdown*. |
| `mixed` | Exibe a primeira ação como botão e agrupa as demais no *dropdown*. |
