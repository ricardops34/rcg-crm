# PoI18nLiterals

**Tipo:** Interface / Modelo
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-i18n-literals

Interface para o método `getLiterals()` do serviço PoI18nService.

## Propriedades

| Propriedade | Tipo | Opcional | Descrição |
|---|---|---|---|
| `context` | `string` | sim | Contexto utilizado na busca das literais. |
| `language` | `string` | sim | Idioma a ser buscado. |
| `literals` | `Array<string>` | sim | Lista das literais. |
