# PoProgressShape

**Tipo:** Enum
**Pacote:** `@po-ui/ng-components`
**Referência:** https://po-ui.io/documentation/po-progress-shape

Enum `PoProgressShape` para definir o formato visual do componente de progresso.

## Valores

| Valor | Descrição |
|---|---|
| `bar` | Formato barra de progresso (padrão). |
| `circle` | Formato circular de progresso. |
